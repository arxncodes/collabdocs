## Common Questions

### Q: How do I change the language?
**A:** You can't change the language after creation. Create a new code document with the desired language.

### Q: Can I work offline?
**A:** No, you need an internet connection for real-time collaboration and auto-save.

### Q: How many people can collaborate?
**A:** Unlimited! Invite as many collaborators as you need.

### Q: Is my code private?
**A:** Yes! Only you and invited collaborators can see your code.

### Q: Can I export my code?
**A:** Yes! Copy the code from the editor and paste it into your local IDE or save it as a file.

## Troubleshooting

### Problem: Editor is blank
**Solution**: Refresh the page or check your internet connection.

### Problem: Syntax highlighting not working
**Solution**: Make sure you selected the correct language when creating the document.

### Problem: Auto-save not working
**Solution**: Check your internet connection and try manual save (Ctrl+S).

### Problem: Can't see collaborators
**Solution**: Make sure they're online and have accepted the invitation.
