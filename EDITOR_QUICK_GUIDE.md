### Change Font
```
1. Select text
2. Click font dropdown
3. Choose font
```

### Change Size
```
1. Select text
2. Click size dropdown
3. Choose size
```

### Change Color
```
1. Select text
2. Click color button
3. Pick color from grid
```

## Common Workflows

### Professional Document
```
1. Title + Subtitle layout
2. Arial 16px body text
3. H1 for sections
4. H2 for subsections
5. Callout boxes for key points
```

### Tutorial/Guide
```
1. Step-by-Step layout
2. Code blocks for examples
3. Warning boxes for gotchas
4. Success boxes for results
5. Highlighted text for tips
```

### Report/Article
```
1. Times New Roman 14px
2. Justified text
3. Quote blocks for citations
4. Two columns for comparisons
5. Headings for structure
```

### Presentation Notes
```
1. Verdana 18px
2. Large headings (32px)
3. Bullet lists
4. Highlighted key points
5. Callout boxes for emphasis
```

## Tips for Best Results

### âœ… Do
- Use consistent fonts throughout
- Maintain readable font sizes (14-18px)
- Use colors sparingly for emphasis
- Left-align most content
- Use layouts for visual interest
- Preview before sharing

### âŒ Don't
- Mix too many fonts (max 2-3)
- Use tiny fonts (<12px)
- Overuse colors
- Center-align body text
- Overuse special layouts
- Forget to save changes

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Formatting not applying | Select text first, then apply |
| Color not showing | Check if in code block |
| Layout not inserting | Click in empty paragraph |
| Font not changing | Try selecting text again |
| Size not updating | Use dropdown, not manual input |
| Alignment not working | Apply to entire paragraph |

## Accessibility Tips

- Use sufficient color contrast
- Don't rely on color alone
- Use headings for structure
- Keep font sizes readable
- Provide text alternatives
- Test with screen readers
